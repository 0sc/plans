DELETE /v1/checklists/:checklist_id/items/:id
Destroy an item

Destroy an item

This endpoint is used to delete a checklist item.

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Invalid token.
404 Item not found.
404 Specified bucketlist not found.

Value:

Must be Integer
id
