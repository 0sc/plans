POST /v1/bucketlists
Create a bucketlist

Create new Bucketlist

This endpoint is used to create a new bucketlist. It requires a name for the new bucketlist.

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Invalid token.
422 Invalid parameter sent

Examples

// Does not create bucketlist if user is not logged in
POST /v1/bucketlists
{
  "bucketlist": {
    "name": "Waylon Daugherty"
  }
}
401
// Does not create bucketlist if name is too long
POST /v1/bucketlists
{
  "bucketlist": {
    "name": "zftu1l54332qzhcbw8xc7myhfnoa0fu7ss8r3uoc57an1ug1hyynpjfzp288shz9xrpkyl2xx837olbpqbfe985orxzxi22yoks78"
  }
}
422
{
  "bucketlists": [
    "Name is too long (maximum is 100 characters)"
  ]
}
// Creates bucketlist with valid params
POST /v1/bucketlists
{
  "bucketlist": {
    "name": "My Bucketlist"
  }
}
201
{
  "bucketlist": {
    "id": 1,
    "name": "My Bucketlist",
    "date_created": "2015-11-30 01:19:18",
    "date_modified": "2015-11-30 01:19:18",
    "created_by": 1,
    "items": []
  }
}
// Does not create bucketlist if params is invalid
POST /v1/bucketlists
{
  "bucketlist": {
    "name": ""
  }
}
422
{
  "bucketlists": [
    "Name can't be blank",
    "Name is too short (minimum is 2 characters)"
  ]
}
// Does not create bucketlist if name is too short
POST /v1/bucketlists
{
  "bucketlist": {
    "name": "o"
  }
}
422
{
  "bucketlists": [
    "Name is too short (minimum is 2 characters)"
  ]
}
