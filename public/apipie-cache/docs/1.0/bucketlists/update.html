Update a Bucketlist

This endpoint is used to update the details of a given bucketlist. It requires a valid user bucketlist id and new name for the bucketlist.

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Invalid token.
422 Invalid parameter sent
400 Empty request params.
404 Bucketlist not found.

Examples

// Returns 422 if params is empty
PATCH /v1/bucketlists/10
{
  "bucketlist": {
    "name": ""
  }
}
422
{
  "bucketlists": [
    "Name can't be blank",
    "Name is too short (minimum is 2 characters)"
  ]
}
// Returns 404 if bucketlist id is invalid
PATCH /v1/bucketlists/100
{
  "bucketlist": {
    "name": "Modified"
  }
}
404
// Returns 400 if bucketlist does not belong to user
PATCH /v1/bucketlists/10
{
  "name": "Modified"
}
400
{
  "bucketlist": {
    "id": 10,
    "name": "f6hj81ury9",
    "date_created": "2015-11-30 01:19:15",
    "date_modified": "2015-11-30 01:19:15",
    "created_by": 1,
    "items": []
  }
}
// Updates the details of a bucketlist with valid params
PATCH /v1/bucketlists/10
{
  "bucketlist": {
    "name": "Modified"
  }
}
200
{
  "bucketlist": {
    "id": 10,
    "name": "Modified",
    "date_created": "2015-11-30 01:19:16",
    "date_modified": "2015-11-30 01:19:16",
    "created_by": 1,
    "items": []
  }
}
// Returns 422 if name is too short
PATCH /v1/bucketlists/10
{
  "bucketlist": {
    "name": "4"
  }
}
422
{
  "bucketlists": [
    "Name is too short (minimum is 2 characters)"
  ]
}
// Returns 422 if name is too long
PATCH /v1/bucketlists/10
{
  "bucketlist": {
    "name": "ks2qjhofx2gnpuk9v7mnodmzsmkzu177obk81tn5jddldhvfqd5e7d90dbbwtwjmrtpixz2vxr2tnji4zcj4pbqjj6tu35d92r4uz"
  }
}
422
{
  "bucketlists": [
    "Name is too long (maximum is 100 characters)"
  ]
}
// Returns 401 if user is not logged in
PATCH /v1/bucketlists/10
{
  "bucketlist": {
    "name": "Modified"
  }
}
401
