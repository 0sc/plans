GET /v1/bucketlists
List bucketlists

== Retrieve Bucketlists

This endpoint is used to retrieve a usersâ€™ bucketlists. It optionally takes params to return the bucketlist in pages and the for limit of bucketlists to fetch per request. It could also be queried with a search params to return all bucketlists that match the params.

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Invalid token.
