GET /v2/bucketlists/:id
Show a bucketlist

Show details of a Bucketlist

This endpoint is used to retrieve more details of a given bucketlist such as created_at. It requires a valid user bucketlist id

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Invalid token.
404 Bucketlist not found.

Examples

// Returns 401 if user is not logged in
GET /v1/bucketlists/1
401
// Returns 404 if bucketlist is not found
GET /v1/bucketlists/1000
404
// Returns 404 if bucketlist is not found
GET /v1/bucketlists/1
404
// Shows the details of a valid bucketlist
GET /v1/bucketlists/1
200
{
  "bucketlist": {
    "id": 1,
    "name": "f6hj81ury9",
    "date_created": "2015-11-30 01:18:30",
    "date_modified": "2015-11-30 01:18:30",
    "created_by": 1,
    "items": []
  }
}

Params

Param name Description
id
required

Id of the user bucketlist to fetch.


Value:

Must be Integer
