POST /v1/bucketlists
Create a bucketlist

== Create new Bucketlist

This endpoint is used to create a new bucketlist. It requires a name for the new bucketlist.

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Invalid token.
422 Invalid parameter sent

Examples

POST /v1/bucketlists
{
  "bucketlist": {
    "name": "My Bucketlist"
  }
}
201
{
  "bucketlist": {
    "id": 1,
    "name": "My Bucketlist",
    "date_created": "2015-11-29 11:30:00",
    "date_modified": "2015-11-29 11:30:00",
    "created_by": 1
  }
}

{

    "name": ""
  }
