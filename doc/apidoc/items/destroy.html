DELETE /v1/checklists/:checklist_id/items/:id
Destroy an item

== Destroy an item

This endpoint is used to delete a checklist item.

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Invalid token.
404 Item not found.
404 Specified bucketlist not found.

Examples

DELETE /v1/bucketlists/1000/items/11
404
DELETE /v1/bucketlists/1000/items/11
404
DELETE /v1/bucketlists/1/items/11
204
DELETE /v1/bucketlists/1/items/100
404
DELETE /v1/bucketlists/1/items/11
404
DELETE /v1/bucketlists/1/items/11
401

Params

Param name Description
checklist_id
required

Id of the checklist item belongs to


Value:

Must be Integer
id
required

Item id


Value:

Must be Integer
