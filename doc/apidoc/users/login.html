POST /auth/login

Authenticate User

This endpoint is used to authenticate a user account.

Supported Formats

json

Errors

422 Query params could not be processed. User not yet created or invalid params provided.

Examples

// Logs user in with valid params and returns auth_token
POST /auth/login
{
  "user": {
    "email": "yeenia@kaulke.name",
    "password": "pass"
  }
}
200
{
  "auth_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzUxMiJ9.eyJ1c2VyIjoxLCJleHAiOjE0NDg5MzI3MDN9.yr8T4TjRufkORvAsV5_OLcxIb9-3Dl0klldyBn2kVr2jwxelGRMgCKCUbhEx647bgNv_6Kjvv_KOj1tYHvtw_w"
}
// Returns 422 if email is not found
POST /auth/login
{
  "user": {
    "email": "floie@becker.info",
    "password": "pass"
  }
}
422
// Return 422 with invalid params
POST /auth/login
{
  "user": {
    "email": "",
    "password": ""
  }
}
