PATCH /users
Update User information

Update User Profile

This endpoint is used to update the user profile details.

Authentication token has to be passed as part of the request. It must be passed as HTTP header(TOKEN).

Supported Formats

json

Errors

401 Unauthorized. Token is probably invalid.
422 Invalid parameter sent
400 Empty request params

Examples

// Returns 422 for request with invalid email
PATCH /users
{
  "user": {
    "name": "Milo Kemmer I",
    "email": "4546$%$",
    "password": "4xbe806agqv"
  }
}
422
{
  "users": [
    "Email is invalid"
  ]
}
// Returns 401 for request with invalid token
PATCH /users
{
  "user": {
    "name": "",
    "email": "yeenia@kaulke.name",
    "password": "password"
  }
}
401
// Returns 422 for request with empty params
PATCH /users
{
  "user": {
    "name": "",
    "email": "",
    "password": ""
  }
}
422
{
  "users": [
    "Name can't be blank",
    "Name is too short (minimum is 2 characters)",
    "Email can't be blank",
    "Email is invalid"
  ]
}
// Returns 400 for request without strong params
PATCH /users
{
  "name": "NewName",
  "email": "new@email.com",
  "password": "newpass"
}
400
{
  "user": {
    "name": "Darlene Sporer",
    "email": "yeenia@kaulke.name",
    "created_at": "2015-11-30T01:18:54.475Z",
    "updated_at": "2015-11-30T01:18:54.499Z"
  }
}
// Returns 200 for request with valid params
PATCH /users
{
  "user": {
    "name": "Eladio Schaefer",
    "email": "email@email.com",
    "password": "uysy4ba18iu"
  }
}
200
{
  "user": {
    "name": "Eladio Schaefer",
    "email": "email@email.com",
    "created_at": "2015-11-30T01:18:54.615Z",
    "updated_at": "2015-11-30T01:18:54.742Z"
  }
}
// Returns 422 if request params is empty
PATCH /users
{
  "user": {
    "name": "",
    "email": "",
    "password": ""
  }
}
422
{
  "users": [
    "Name can't be blank",
    "Name is too short (minimum is 2 characters)",
    "Email can't be blank",
    "Email is invalid"
  ]
}
